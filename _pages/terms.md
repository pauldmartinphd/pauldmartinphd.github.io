This website does not collect personal information from visitors. It sets no cookies, runs no advertising, includes no analytics or tracking scripts, and hosts no comment system.

The site is served through GitHub Pages. As with any web host, GitHub's servers may record standard technical request data — such as IP address, browser type, and referring page — in server logs for security and operational purposes. That data is handled under [GitHub's Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement); this site's author does not have access to it and does not use it.

Questions about this policy may be directed to [pmartin@pauldmartin.org](mailto:pmartin@pauldmartin.org).
