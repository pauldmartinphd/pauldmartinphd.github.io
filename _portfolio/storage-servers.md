Designed and built multiple high-capacity storage server platforms based on ZFS, using repurposed enterprise hardware including HP MediaSmart EX470, Dell T5400, Rackable Systems chassis, and custom builds. These systems provide network-attached storage, backup infrastructure, and media serving with ZFS features including data integrity verification, snapshots, replication, and flexible RAID configurations.
