Multiple projects replacing proprietary BIOS and UEFI firmware with the open-source CoreBoot firmware on various platforms. This includes ThinkPad T420/T430/T520/T530 systems, where replacement requires direct SPI flash chip programming and careful firmware configuration to support all hardware components, as well as Chromebook modifications to replace the stock CoreBoot configuration with custom builds enabling full Linux support.

Related work includes BIOS-level serial number programming, Frankenpad builds combining components across ThinkPad models (e.g., X61 screen, X60 casing, X61 tablet motherboard with PLL overclocking and modified BIOS), and custom BIOS modifications for hardware feature enablement.
