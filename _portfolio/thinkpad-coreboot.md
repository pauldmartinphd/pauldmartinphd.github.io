Multiple projects replacing proprietary BIOS and UEFI firmware with the open-source Coreboot firmware. This includes ThinkPad systems requiring direct SPI flash chip programming, careful firmware configuration to support all hardware components, and BIOS-level serial number programming. Related work includes custom BIOS modifications for hardware feature enablement and PLL overclocking through firmware modification.

This experience with low-level firmware structures, flash programming, and boot chain analysis directly informs expert witness work involving firmware disputes and embedded system investigations.
