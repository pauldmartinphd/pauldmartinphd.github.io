Replaced the proprietary BIOS on a ThinkPad T420 with the open-source CoreBoot firmware, requiring direct hardware interaction with the SPI flash chip and careful configuration of the firmware build to support all hardware components. Related work includes BIOS-level serial number programming on ThinkPad T420/T430/T520/T530 systems and Frankenpad builds combining components across ThinkPad models (e.g., X61 screen, X60 casing, X61 tablet motherboard with PLL overclocking and modified BIOS).
